# 许可证分析接口使用说明

创建生成任务

POST `/api/license/v1/metrics/generate`  
完整 url：`http://127.0.0.1:8002/api/license/v1/metrics/generate`

请求参数：

示例：

字段 | 类型 | 说明
--- | --- | ---
repo_url | string | github 仓库地址

```json
{
  "repo_url": "https://github.com/pytorch/pytorch"
}
```

返回结果

```json
{
  "status": "submitted",
  "message": "License profiling task submitted successfully",
  "task_id": "task_9e6b0d77",
  "repo_name": "pytorch/pytorch",
  "repo_id": 2716301348,
  "upstream_status": 200,
  "timestamp": "2026-03-13T09:10:00Z"
}
```

字段说明

字段 | 类型 | 说明
--- | --- | ---
status | string | 任务提交状态
message | string | 任务提交消息
task_id | string | 任务唯一 ID
repo_name | string | 仓库名称
repo_id | integer | 仓库 ID（当前服务内生成的整数标识）
upstream_status | integer | 任务服务返回状态
timestamp | string | 接口返回时间（UTC ISO8601 格式）

curl 示例：

```bash
curl -X POST "http://127.0.0.1:8002/api/license/v1/metrics/generate" \
  -H "Content-Type: application/json" \
  -d '{"repo_url":"https://github.com/pytorch/pytorch"}'
```

查询任务状态

GET `/api/license/v1/task/{task_id}`  
完整 url：`http://127.0.0.1:8002/api/license/v1/task/{task_id}`

路径参数

参数 | 类型 | 说明
--- | --- | ---
task_id | string | 创建任务时返回的任务 ID

示例：`task_9e6b0d77`

请求示例：

```bash
curl "http://127.0.0.1:8002/api/license/v1/task/task_9e6b0d77"
```

返回示例：

字段 | 类型 | 说明
--- | --- | ---
status | string | 接口调用状态，`success` 表示接口调用成功
task_id | string | 任务唯一 ID
task_status | string | 任务执行状态
timestamp | string | 接口返回时间（UTC ISO8601 格式）
error | string | 任务执行失败时返回的错误信息（仅在 `failed` 状态时出现）

任务执行中：

```json
{
  "status": "success",
  "task_id": "task_9e6b0d77",
  "task_status": "submit",
  "timestamp": "2026-03-13T09:10:02Z"
}
```

任务成功：

```json
{
  "status": "success",
  "task_id": "task_9e6b0d77",
  "task_status": "success",
  "timestamp": "2026-03-13T09:10:30Z"
}
```

任务失败：

```json
{
  "status": "success",
  "task_id": "task_9e6b0d77",
  "task_status": "failed",
  "error": "...",
  "timestamp": "2026-03-13T09:10:20Z"
}
```

任务不存在：

```json
{
  "status": "success",
  "task_id": "task_xxx",
  "task_status": "not_found",
  "timestamp": "2026-03-13T09:10:05Z"
}
```

获取生成结果接口

GET `/api/license/v1/result/{task_id}`  
完整 url：`http://127.0.0.1:8002/api/license/v1/result/{task_id}`

路径参数

参数 | 类型 | 说明
--- | --- | ---
task_id | string | 创建任务时返回的任务 ID

示例：`task_9e6b0d77`

请求示例：

```bash
curl -sS "http://127.0.0.1:8002/api/license/v1/result/task_9e6b0d77" | jq .
```

返回示例（任务成功）：

```json
{
  "status": "success",
  "task_id": "task_9e6b0d77",
  "task_status": "success",
  "repo_name": "pytorch/pytorch",
  "repo_id": 2716301348,
  "repo_url": "https://github.com/pytorch/pytorch",
  "results": {
    "license:": ["bsd-3-clause"],
    "file_number": 1234,
    "license_kind:": 9,
    "license_total:": 56,
    "license_conflict:": []
  },
  "timestamp": "2026-03-13T09:10:30Z"
}
```

说明：`results` 字段为当前仓库许可证分析模块真实返回结果，字段名与旧文档可能不完全一致，以接口实际返回为准。

查询接口

POST `/api/license/v1/metrics/query`  
完整 url：`http://127.0.0.1:8002/api/license/v1/metrics/query`

请求参数：

字段 | 类型 | 说明
--- | --- | ---
repo_url | string | github 仓库地址

示例：

```json
{
  "repo_url": "https://github.com/pytorch/pytorch"
}
```

curl 示例：

```bash
curl -sS -X POST "http://127.0.0.1:8002/api/license/v1/metrics/query" \
  -H "Content-Type: application/json" \
  -d '{"repo_url":"https://github.com/pytorch/pytorch"}' | jq .
```

返回示例（已存在结果）：

```json
{
  "status": "success",
  "repo_url": "https://github.com/pytorch/pytorch",
  "task_id": "task_9e6b0d77",
  "task_status": "success",
  "repo_name": "pytorch/pytorch",
  "repo_id": 2716301348,
  "results": {},
  "timestamp": "2026-03-13T09:10:30Z"
}
```

返回示例（结果不存在）：

```json
{
  "status": "success",
  "repo_url": "https://github.com/pytorch/pytorch",
  "result_status": "not_found",
  "timestamp": "2026-03-13T09:11:00Z"
}
```

查询许可证分析结果仓库数量

GET `/api/license/v1/metrics/repos/count`  
完整 url：`http://127.0.0.1:8002/api/license/v1/metrics/repos/count`

请求参数：无

请求示例：

```bash
curl "http://127.0.0.1:8002/api/license/v1/metrics/repos/count"
```

返回结果示例：

```json
{
  "status": "success",
  "repo_count": 128,
  "timestamp": "2026-03-13T09:12:00Z"
}
```

返回示例：

字段 | 类型 | 说明
--- | --- | ---
status | string | 接口调用状态，`success` 表示接口调用成功
repo_count | integer | 已生成许可证分析结果的仓库数量
timestamp | string | 接口返回时间（UTC ISO8601 格式）

模糊搜索许可证分析结果仓库列表

GET `/api/license/v1/metrics/repos/search`  
完整 url：`http://127.0.0.1:8002/api/license/v1/metrics/repos/search`

请求参数：

字段 | 类型 | 说明
--- | --- | ---
keyword | string | 仓库名称关键词

请求示例：

```bash
curl "http://127.0.0.1:8002/api/license/v1/metrics/repos/search?keyword=pytorch/p"
```

返回结果：

字段 | 类型 | 说明
--- | --- | ---
status | string | 接口调用状态，`success` 表示接口调用成功
keyword | string | 搜索关键词
count | integer | 本次返回的仓库数量
repos | array | 仓库结果列表
repos[].repo_id | integer | 仓库 ID
repos[].repo_name | string | 仓库名称，格式一般为 owner/repo
repos[].url | string | 仓库 github 地址
timestamp | string | 接口返回时间（UTC ISO8601 格式）

注意：该接口返回的模糊搜索仓库列表最多为 200 条。  
当搜索匹配结果超过 200 条时，仅返回前 200 条；当匹配结果少于 200 条时，返回全部匹配结果。

返回结果示例：

```json
{
  "status": "success",
  "keyword": "pytorch/p",
  "count": 2,
  "repos": [
    {
      "repo_id": 2716301348,
      "repo_name": "pytorch/pytorch",
      "url": "https://github.com/pytorch/pytorch"
    },
    {
      "repo_id": 1656060482,
      "repo_name": "pytorch/pytorch-probot",
      "url": "https://github.com/pytorch/pytorch-probot"
    }
  ],
  "timestamp": "2026-03-13T09:13:00Z"
}
```
